### Install: (Run in Terminal)
```bash
cp .env.example .env
composer install 	
# or: composer install --ignore-platform-reqs
# or: composer update 
# or: composer update --ignore-platform-reqs  
npm install
npm audit fix
npm run dev
# create database: 
php artisan migrate:fresh --seed
php artisan key:generate
#Web 
php artisan serve --port=8001
#Api
php artisan serve
```
